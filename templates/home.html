The fastest driver's test booking bot in Canada.

Get alerts for the next cancellation

Everyday, there is always a handful number of people cancelling their driver's test at every location.

Run this bot and become notified whenever someone does, to reserve your own test!

Technologies Used:

Python

Used to create the script's main logic and alert system with it's object oriented programming principles.

Selenium

Used for automation and web scraping inside Python.

Flask

Serves as a back-end for the application; stores the script while also providing an opportunity for a front-end UI.

Bootstrap

Alongside HTML, CSS, and Javascript, forms the user friendly front-end of this application.
